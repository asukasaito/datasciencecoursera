Overview

In this document, Analysis of NYPD shooting incidents is conducted. To see the big picture, time series analysis on the number of incidents and relation between the number of indicants and perpetrators race are conducted.

Dataset

The data set contain data from 2006 to 2022 about shooting incidents happened in NY with attributes of Perpetrators and victims. Data set can be found here. And detail headers (columns) will be shonwn below. https://catalog.data.gov/dataset/nypd-shooting-incident-data-historic

Structure

This documents consists of 4 parts; 1) preparation, 2) time series analysis, 3) analysis by race and 4) regression by race.

Trial to predict the number of insidents from race and year

Graphical analysis is not always enough to show evidence of correlation. Here statistical analysis is conducted to see it by quantitatively. As it is countable, Poisssion distribution is used with generalized linear regression mode.

model = glm( INCIDENT_KEY ~ year+PERP_RACE, data = df.gb2, family =poisson )

Interpretation

The summary of the Poisson regression model shows that each factor, such as year and race, is statistically significant. However, the plots indicate that the residuals are not normally distributed; for instance, the QQ plot shows a light tail. This suggests that the model is not a good fit. Using Poisson regression would be more reasonable for predicting countable datasets. Therefore, a more reasonable conclusion would be that the parameters are not sufficient.
